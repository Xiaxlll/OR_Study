## 构造方法
1. 定义`Person`类，并创造一个`Person`对象`zs`
``` java
class Person {
  String name;
  int age;
}

public class Main {
  public static void main(String[] args) {
    Person zs = new Person();
    zs.name = "张三";
    zs.age = 18;
    System.out.println("姓名：" + zs.name);
    System.out.println("年龄：" + zs.age);
  }
}
```
