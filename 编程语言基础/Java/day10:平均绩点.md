## 扩展：`Switch-Case`
``` java
switch (expression) {
    case value1:
        // 与 value1 匹配时执行的代码
        break;
    case value2:
        // 与 value2 匹配时执行的代码
        break;
    // 可以有多个 case 分支
    default:
        // 如果没有匹配的 case 分支，执行 default 分支
}
```
* 注意：`case` 中不能使用变量，而不是一个确定的值
